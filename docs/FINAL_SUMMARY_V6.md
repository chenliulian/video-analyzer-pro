## 🔧 技术要点

### 1. OCR文件格式

```
【frame_0003_15.00s.jpg】
----------------------------------------------------------------------
我的白鸽
年    级：七年级                学    科：语文（统编版）
```

**解析关键**:
- 图片标记: `【filename】`
- 分隔线: `------` (需跳过)
- 路径匹配: basename → 完整路径

### 2. 时间匹配算法

```python
# 区间重叠判断
if not (seg_end < start_time or seg_start > end_time):
    matching_segments.append((seg_start, seg_end, seg_text))
```

### 3. Fallback机制

```python
# 优先LLM精炼结果
if refined_results and img_path in refined_results:
    refined_text = refined_results[img_path].strip()
    if refined_text:  # 确保非空
        return refined_text

# Fallback到原始转录
matching_segments = [...]
return " ".join([text for _, _, text in matching_segments])
```
