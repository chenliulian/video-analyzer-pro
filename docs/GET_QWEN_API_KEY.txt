===============================================================================
如何获取阿里云通义千问 API Key
===============================================================================

问题：
  ❌ Invalid API-key provided
  
原因：
  .env 文件中的 DASHSCOPE_API_KEY 是占位符，不是真实的 API Key

===============================================================================
解决方案（3步获取 API Key）
===============================================================================

步骤1: 访问阿里云百炼平台
  🔗 https://bailian.console.aliyun.com/

步骤2: 登录/注册
  • 支付宝快捷登录（推荐）
  • 淘宝账号登录
  • 手机号注册

步骤3: 创建 API Key
  1. 登录后，点击右上角头像
  2. 选择 "API Key 管理"
  3. 点击 "创建新的 API Key"
  4. 复制生成的 API Key（格式: sk-xxxxxxxxxxxxxxxx）
  
  ⚠️  注意：API Key 只显示一次，请立即保存！

===============================================================================
配置 API Key
===============================================================================

方法1: 编辑 .env 文件（推荐）

  nano .env
  
  然后修改：
  DASHSCOPE_API_KEY=sk-xxxxxxxxxxxxxxxx  # 替换为你的真实API Key

方法2: 直接在代码中传入

  from ocr_qwen import QwenOCR
  
  ocr = QwenOCR(api_key="sk-xxxxxxxxxxxxxxxx")  # 你的真实API Key
  text = ocr.recognize_image("image.jpg")

===============================================================================

===============================================================================

测试1: 检查环境变量
  python3 -c "import os; from dotenv import load_dotenv; load_dotenv(); print(os.getenv('DASHSCOPE_API_KEY'))"

  python3 ocr_qwen.py --image extracted_frames/frame_0012_290.00s.jpg

预期输出：
  ✓ Qwen OCR 初始化成功
  模型: qwen-vl-max
  
  识别结果:
  ----------------------------------------------------------------------
  (图片中的文字内容)

===============================================================================
常见问题
===============================================================================

Q: 没有阿里云账号怎么办？
A: 支持支付宝快捷注册，1分钟完成

Q: API Key 有免费额度吗？
A: 有！新用户通常有免费试用额度

Q: API Key 泄露了怎么办？
A: 立即在控制台删除旧Key，创建新的

Q: 可以用 LLM_API_KEY 吗？
A: 不可以！那是用于文本精炼的，OCR需要单独的 DASHSCOPE_API_KEY
