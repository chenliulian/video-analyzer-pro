# Word文档导出功能使用指南

## 功能简介

视频分析器Pro版现在支持同时生成**PDF**和**Word文档**两种格式的输出:
- **PDF**: 包含图片和文字的完整报告(不可编辑)
- **Word**: 可编辑的文档,方便用户修改文字内容

## 安装依赖

```bash
pip3 install python-docx
```

## 使用方法

### 1. 基本用法

```bash
# 使用Whisper ASR (推荐,稳定可靠)
python3 video_analyzer_pro.py --video your_video.mp4

# 使用Qwen ASR (支持大文件自动切分)
python3 video_analyzer_pro.py --video your_video.mp4 --asr-engine qwen
```

### 2. 输出文件

程序会在 `results/` 目录下生成以下文件:

```
results/
├── {video_id}_frames_pro.pdf      # PDF报告
├── {video_id}_frames_pro.docx     # Word文档 ⭐新增
├── {video_id}_transcript.txt      # 语音转录
├── {video_id}_refined.json        # LLM精炼结果
├── {video_id}_ocr_qwen.txt        # OCR识别结果
└── extracted_frames/               # 提取的关键帧
```

## Word文档结构

### 文档布局

1. **封面信息**
   - 视频分析报告标题
   - 生成时间
   - 视频文件名
   - 总帧数

2. **每一帧包含**
   - 帧标题 (帧序号 + 时间范围)
   - 课件截图 (居中显示,宽度6英寸)
   - 文字内容标题
   - 转录/精炼后的文字内容

## 编辑Word文档

### 支持的操作

✅ **文字编辑**: 可以直接修改、添加、删除文字内容
✅ **格式调整**: 可以修改字体、字号、颜色等
✅ **图片操作**: 可以调整图片大小、位置
✅ **内容重组**: 可以复制、移动、删除章节
✅ **导出其他格式**: 可以另存为PDF、HTML等

### 推荐软件

- **Microsoft Word** (Windows/Mac)
- **WPS Office** (跨平台,免费)
- **LibreOffice Writer** (开源,免费)
- **Google Docs** (在线,需上传)

## Qwen ASR 大文件处理

### 文件大小限制

Qwen ASR对单个文件有大小限制(约8-10 MB),本程序已实现**自动切分功能**:

- 检测文件大小
- 自动切分为2分钟的小段
- 压缩音频(单声道,16kHz,64kbps)
- 逐段调用API转录
- 合并所有转录结果

### 处理时间

- 每2分钟音频段约需10-20秒
- 20分钟视频大约需要2-3分钟

## 常见问题

### Q1: Word文档打不开?

**A**: 确保安装了 `python-docx`:
```bash
pip3 install python-docx
```

### Q2: Word文档中文字乱码?

**A**: 使用支持UTF-8的编辑器打开,推荐使用Microsoft Word或WPS。

### Q3: 只想生成Word,不想生成PDF?

**A**: 目前两者是一起生成的,如需单独生成Word,可以使用临时脚本:

```python
from video_analyzer_pro import VideoAnalyzerPro
# ... (参考 generate_word.py)
```

### Q4: Qwen ASR转录失败?

**A**: 检查以下几点:
1. API Key是否配置正确
2. 网络是否可以访问新加坡API (`--asr-region intl`)
3. 查看错误日志,确认具体原因

### Q5: 如何提高转录质量?

**A**: 
- **Whisper**: 使用更大的模型 `--whisper-model medium` 或 `large`
- **Qwen ASR**: 启用ITN `--asr-enable-itn` (数字、日期格式化)
- **LLM精炼**: 不要使用 `--no-llm`,让LLM纠错优化文字

## 性能对比

| 引擎 | 优点 | 缺点 | 推荐场景 |
|------|------|------|----------|
| **Whisper** | 无文件大小限制<br>离线运行<br>稳定可靠 | 需要GPU加速<br>模型较大 | 大文件<br>离线环境 |
| **Qwen ASR** | 在线API<br>速度快<br>准确率高 | 文件大小限制<br>需要网络 | 小文件<br>在线环境 |
