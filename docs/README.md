# 视频分析工具

## 功能说明
这个工具可以对视频文件进行全面分析:
1. 从视频文件中提取音频
2. 使用 OpenAI 的 Whisper 模型将音频转换为文字
3. **智能提取视频关键帧**(使用场景变化检测)
4. 将提取的图片整合成PDF文档

## 性能优化 ⚡
- ✅ 使用 **OpenCV** 进行高效图像处理
- ✅ 使用 **iter_frames** 代替 get_frame,大幅提升处理速度
- ✅ 智能场景变化检测,自动识别画面切换
- ✅ 避免重复和相似帧,提取关键内容

## 安装依赖

首先确保系统安装了 ffmpeg:
```bash
# macOS
brew install ffmpeg

# Linux (Ubuntu/Debian)
sudo apt-get install ffmpeg

# Linux (CentOS/RHEL)
sudo yum install ffmpeg
```

然后安装 Python 依赖:
```bash
pip3 install -r requirements.txt
```

## 使用方法

### 方式一: 使用整合工具(推荐)
一键完成音频转文字和图片提取:
```bash
python3 video_analyzer.py
```

### 方式二: 单独使用各个功能

**仅提取音频并转文字:**
```bash
python3 extract_audio_to_text.py
```

**仅提取关键帧并生成PDF:**
```bash
python3 extract_frames_to_pdf.py
```

### 方式三: 测试和优化

**运行相似度算法测试(推荐在处理前运行):**
```bash
python3 test_similarity.py
```

这个工具会:
- 测试5种不同的相似度计算方法
- 分析当前提取帧的质量
- 推荐最优阈值设置
- 帮助你为不同类型视频找到最佳参数

## 输出文件

- `视频名.mp3` - 提取的音频文件
- `视频名_transcript.txt` - 转录的文字内容(包含完整文本和带时间戳的分段文本)
- `extracted_frames/` - 提取的关键帧图片文件夹
- `视频名_frames.pdf` - 整合所有关键帧的PDF文档

## 配置说明

### 音频转文字
可以在脚本中修改 `model_size` 参数来选择不同的 Whisper 模型:
- `tiny` - 最快,准确度较低
- `base` - 平衡选择(默认)
- `small` - 更准确,速度稍慢
- `medium` - 高准确度,较慢
- `large` - 最高准确度,最慢

### 关键帧提取(优化版)
可以在脚本中调整以下参数:
- `check_interval` - 检查间隔(秒),默认1秒检查一次
- `scene_threshold` - 场景变化阈值,默认30,值越大过滤越严格
- `min_interval` - 同一场景最小保存间隔(秒),默认3秒

**参数推荐**:
- 教学视频(PPT): `threshold=30, min_interval=3.0`
- 演讲视频: `threshold=35, min_interval=5.0`  
- 操作演示: `threshold=25, min_interval=2.0`
- 严格去重: `threshold=40, min_interval=5.0`

💡 **提示**: 运行 `python3 test_similarity.py` 可以测试并找到最适合你视频的参数

- ✅ 保留所有关键画面变化

## 注意事项

- 首次运行时会自动下载 Whisper 模型(base模型约 140MB)
- 处理时间取决于视频长度和所选参数
- 支持中文语音识别
- 关键帧提取会自动过滤相似帧,避免重复
- 运行前会自动清空 `extracted_frames` 文件夹
