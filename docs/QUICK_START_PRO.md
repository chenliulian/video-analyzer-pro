# 视频分析器Pro版 - 5分钟快速开始

## 🎯 目标

将教育讲课视频一键转换为完整的PDF文档(图片+文字)

## 📋 准备工作

### 1. 安装依赖

```bash
pip3 install moviepy pillow opencv-python openai-whisper torch dashscope python-dotenv
```

### 2. 配置API密钥

创建 `.env` 文件:

```bash
# 必需 - 用于OCR识别
DASHSCOPE_API_KEY=sk-your-qwen-api-key

# 可选 - 用于文字精炼
LLM_API_KEY=your-openai-api-key
```

**获取通义千问API密钥:** https://help.aliyun.com/zh/model-studio/get-api-key

## 🚀 开始使用

### 最简单的用法

```bash
python3 video_analyzer_pro.py --video your_lecture.mp4
```

就这么简单! 脚本会自动完成:

✅ 提取关键帧  
✅ 提取音频  
✅ 语音转文字  
✅ OCR识别  
✅ 生成PDF  

### 输出结果

```
results/
├── extracted_frames/          # 提取的关键帧
├── your_lecture.mp3          # 提取的音频
├── your_lecture_transcript.txt   # 语音转录
├── your_lecture_ocr_qwen.txt    # OCR识别结果
└── your_lecture_frames_pro.pdf  # 最终PDF ⭐
```



### 断点续传

如果处理中断,再次运行相同命令,会自动跳过已完成的步骤:

```bash
# 第一次运行(中断)
python3 video_analyzer_pro.py --video lecture.mp4
# Ctrl+C 中断

# 第二次运行(自动继续)
python3 video_analyzer_pro.py --video lecture.mp4
# ✓ 发现已提取的帧 (跳过)
# ✓ 发现已提取的音频 (跳过)
# ⏳ 继续后续步骤...
```

## ❓ 常见问题

### Q: 首次运行为什么慢?

**A:** 首次运行需要下载Whisper模型(~1-2GB),之后会很快。

### Q: 可以不使用LLM吗?

**A:** 可以! 使用 `--no-llm` 参数即可跳过,只做OCR和转录。

### Q: API调用费用是多少?

**A:** 
- OCR: 约 ¥0.01-0.05/张图片
- LLM: 约 ¥0.1-0.5/次精炼
- 10分钟视频估计成本: ¥1-5元

### Q: 出错了怎么办?

**A:** 
1. 检查 `.env` 文件中的API密钥
2. 确保网络连接正常
3. 查看终端输出的详细错误信息
4. 使用 `--no-skip` 强制重新处理
